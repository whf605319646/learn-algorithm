# 面试题_判断括号字符串是否有效

![20201029_223700_36](image/20201029_223700_36.png)

## 堆栈解法

1. 对于左括号，肯定是合法的，需要判断是否有右括号匹配，留着就是放到堆栈中，存着，等待右括号
2. 对于右括号，需要与左括号匹配，女人就得找男人。如果匹配就得弹出栈中的左括号
3. 最终走完，栈必须是空的

  - 如果是右括号，就要匹配栈顶
